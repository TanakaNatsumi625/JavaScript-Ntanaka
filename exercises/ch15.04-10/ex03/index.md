### 開発者ツールで CSS に関して実行できる操作
- 参考(Googleの説明がわかりやすかった)：https://developer.chrome.com/docs/devtools/css?hl=ja
1. クラスを追加する
    - クラスを追加することでどの属性がどのように聞いているのかを、開発者ツールベースで検証できるのは便利と感じた
    - 特に初心者だと、クラスを追加して属性を消したり付けたりして影響範囲を確認しやすいと感じた
2. ホバー時の動作を確認する
    - カーソルを合わせてホバーの動作を確認する動作は煩雑ではないが、コンソールでチェックを付けるだけでいちいちカーソルを合わせなくてもイメージしやすくなるのは便利と感じた
3. 要素のサイズを変更する
    - 個人的に一番CSSで難しいのがmarginやpaddingだと感じているので、実際ブラウザでどのように表示されるかをコンソールベースでできるのは便利

### Tailwind CSS のクラス を開発者ツールから追加すると変更が反映されない理由
- tailwind CSSは`npx tailwindcss -i ./ex02/input.css -o ./ex02/style.css`で作成したstyle.cssしか参照しない。
- よって、開発者ツールで後からクラスを足しても、そもそもstyle.cssに存在しないので変更は反映されない
